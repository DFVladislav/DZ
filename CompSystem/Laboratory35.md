# Освоение Midnight Commander: работа с файлами и папками в Linux

**Цель занятия** — научиться пользоваться программой Midnight Commander (MC) для управления файловой системой в операционной системе GNU/Linux.

## Рассматриваемые вопросы
1. Первое знакомство с Midnight Commander.
2. Настройка отображения содержимого панелей.
3. Различные способы сортировки.
4. Просмотр и редактирование текстовых файлов, создание каталогов.
5. Копирование файлов и папок.
6. Перемещение и переименование.
7. Удаление ненужных объектов.
8. Создание символических ссылок.
9. Определение занятого и свободного места на диске.
10. Подключение к FTP-серверам.
11. Подключение к SFTP-серверам через SSH.
12. Использование встроенной командной строки.

## Оборудование и подготовка

Работа выполняется на компьютерах с бесплатной операционной системой Ubuntu 12.04 LTS (или Debian 7) и графической средой GNOME.

Обратитесь к преподавателю для включения компьютера. Войдите в систему, используя выданные логин и пароль.

## 1. Запуск Midnight Commander

Откройте терминал: нажмите `<Ctrl+Alt+T>` или выберите в меню `Applications → Accessories → Terminal`. Наберите команду `mc` и нажмите Enter. Разверните окно на весь экран.

Изучите интерфейс: две панели, верхнее меню, подсказки по функциональным клавишам внизу.

Отчёт оформляйте в LibreOffice Writer (`Applications → Office → LibreOffice Writer`). Сделайте скриншот окна MC и сохраните его в отчёте.

## 2. Настройка отображения

Выполните последовательно следующие действия.

- **Режим «Дерево».** В меню `Right` выберите `Tree` (или нажмите `<F9>`, затем `<R>`, затем `<T>`). Посмотрите структуру каталога `/etc`. Сделайте скриншот.
- **Информация об объекте.** В меню `Right` выберите `Info` (`<F9>`, `<R>`, `<Ctrl+x>`, `<i>`). На левой панели выделите любой файл или папку – справа появятся сведения о нём. Скриншот.
- **Быстрый просмотр.** В меню `Right` выберите `Quick View` (`<F9>`, `<R>`, `<Ctrl+x>`, `<q>`). На левой панели откройте, например, `/etc/fstab` – его содержимое отобразится справа. Скриншот.
- **Обычный список.** Вернитесь к стандартному режиму: `Right` → `File listing` (`<F9>`, `<R>`, `<g>`). Скриншот.

## 3. Сортировка объектов

- **По размеру.** Откройте `Right` → `Sort order` (`<F9>`, `<R>`, `<S>`). Выберите `Size` (клавиша `<S>`), нажмите `OK`. Перейдите в `/etc`. Самый большой файл окажется внизу списка. Скриншот.
- **По времени изменения.** В том же окне выберите `Modify Time` (клавиша `<m>`), `OK`. Внизу будет файл с самой свежей датой. Скриншот.
- **По имени.** Выберите `Name` (клавиша `<N>`), `OK`. В конце списка окажется файл, начинающийся с буквы, близкой к концу алфавита (например, `z`). Скриншот.

## 4. Просмотр, редактирование, создание папок

- **Просмотр файла.** Выделите текстовый файл (например, `/etc/fstab`) и нажмите `<F3>` (или `File` → `View`). Скриншот.
- **Создание нового файла.** Перейдите в домашний каталог. Нажмите `<Shift+F4>`. Наберите любой текст. Сохраните файл под именем `newfile` клавишей `<F2>`, затем закройте редактор `<F10>`. Откройте этот файл снова, нажав `<F4>`. Скриншот.
- **Создание папки.** Нажмите `<F7>` (или `File` → `Mkdir`), введите имя `new_dir`, нажмите `OK`. Установите курсор на созданную папку – скриншот.

## 5. Копирование

Midnight Commander очень удобен для копирования целых каталогов. Одна панель – исходная, другая – целевая. Переключайтесь между панелями мышью или клавишей `<Tab>`.

Скопируйте каталог `/etc` в `/tmp`. На левой панели откройте корневой каталог `/`, на правой – `/tmp`. Выделите на левой панели папку `etc` и нажмите `<F5>` (или `File` → `Copy`). В открывшемся окне обратите внимание на флажок `Preserve attributes` (сохранять атрибуты – аналог `cp -p`). Нажмите `OK`. Дождитесь окончания копирования. На правой панели перейдите в скопированную папку и сделайте скриншот.

## 6. Перемещение и переименование

На обеих панелях откройте `/tmp`. Выделите папку `etc` и нажмите `<F6>` (или `File` → `Rename/Move`). Она переместится на правую панель. При желании переименуйте её в `etc_new`. Скриншот.

## 7. Удаление

Выделите ненужный объект (например, папку `etc_new`) и нажмите `<F8>` (или `File` → `Delete`). Подтвердите удаление: сначала `Yes`, затем в следующем окне для рекурсивного удаления всего содержимого выберите `All`. По окончании сделайте скриншот.

## 8. Поиск файлов и папок

Поиск вызывается через меню `Command` → `Find file` (`<F9>`, `<C>`, `<F>`).

- **Поиск по маске.** Перейдите в домашний каталог. В поле `File name:` введите `*.png`, снимите галочку `Search for content`. Нажмите `OK`. После завершения поиска сделайте скриншот с результатами.
- **Поиск текста внутри файлов.** Перейдите в `/etc`. В поле `File name:` укажите `*`, поставьте галочку `Search for content`, а в поле `Content:` введите `clock`. Нажмите `OK`. Скриншот результатов.

## 9. Создание символических ссылок

Перейдите в `/tmp` на обеих панелях. Выделите любой файл. Нажмите `<F9>`, затем `<F>`, затем `<S>` (или `File` → `Symlink`). В строке `Symbolic link filename:` допишите в конец `_link` и нажмите `OK`. Обратите внимание – перед именем ссылки появится символ `@`. Скриншот. Установите курсор на ссылку – внизу панели отобразится путь к исходному файлу.

## 10. Оценка занятого и свободного места

- В правом нижнем углу каждой панели показано `Доступно / Всего` (в байтах и процентах).
- Чтобы узнать размер каталогов, выберите `Command` → `Show directory sizes` (или нажмите `<Ctrl+Space>`). Перейдите в `/etc`, отсортируйте по размеру, затем выполните эту команду – вы увидите занимаемое место. Скриншот.

## 11. Подключение к FTP-серверу

В Midnight Commander есть встроенный FTP-клиент. Выберите `Left` → `FTP link` (`<F9>`, `<L>`, `<P>`). В появившемся окне введите `ftp://mirror.yandex.ru` и нажмите `OK`. На активной панели отобразится содержимое публичного FTP-сервера. Сделайте скриншот.

## 12. Подключение к SFTP-серверу по SSH

Выберите `Left` → `Shell link` (`<F9>`, `<L>`, `<S>`). Введите адрес `server` и нажмите `OK`. На панели появится корневой каталог удалённого сервера. Скриншот.

## 13. Использование встроенной командной строки

В нижней части окна MC всегда есть приглашение командной строки. Введите команду `date` и нажмите Enter. Затем нажмите `<Ctrl+O>` – Midnight Commander временно скроется, и вы увидите вывод команды. Сделайте скриншот. Вернитесь в MC повторным нажатием `<Ctrl+O>`.

Покажите готовый отчёт преподавателю.

## Ответы на контрольные вопросы

1. **Как запустить Midnight Commander?**  
   Открыть терминал и ввести команду `mc`.

2. **Как посмотреть структуру каталога в виде дерева?**  
   В меню `Right` выбрать `Tree` (или `<F9>`, `<R>`, `<T>`).

3. **Как вывести информацию о выбранном объекте?**  
   В меню `Right` выбрать `Info` (`<F9>`, `<R>`, `<Ctrl+x>`, `<i>`).

4. **Как включить быстрый просмотр содержимого файла?**  
   В меню `Right` выбрать `Quick View` (`<F9>`, `<R>`, `<Ctrl+x>`, `<q>`).

5. **Как переключить панель в обычный режим списка файлов?**  
   В меню `Right` выбрать `File listing` (`<F9>`, `<R>`, `<g>`).

6. **Как отсортировать объекты по размеру?**  
   `Right` → `Sort order` → выбрать `Size` (`<S>`) → `OK`.

7. **Как отсортировать объекты по дате изменения?**  
   `Right` → `Sort order` → выбрать `Modify Time` (`<m>`) → `OK`.

8. **Как отсортировать объекты по имени?**  
   `Right` → `Sort order` → выбрать `Name` (`<N>`) → `OK`.

9. **Как просмотреть файл во встроенном просмотрщике?**  
   Выделить файл и нажать `<F3>` (или `File` → `View`).

10. **Как создать новый файл?**  
    Нажать `<Shift+F4>`, набрать текст, сохранить (`<F2>`), закрыть (`<F10>`).

11. **Как редактировать файл во встроенном редакторе?**  
    Выделить файл и нажать `<F4>` (или `File` → `Edit`).

12. **Как создать новый каталог?**  
    Нажать `<F7>` (или `File` → `Mkdir`), ввести имя.

13. **Как скопировать объекты с одной панели на другую?**  
    Выделить объект, нажать `<F5>` (или `File` → `Copy`).

14. **Как переместить или переименовать объекты?**  
    Выделить объект, нажать `<F6>` (или `File` → `Rename/Move`).

15. **Как удалить объекты?**  
    Выделить объект, нажать `<F8>` (или `File` → `Delete`), подтвердить.

16. **Как найти объект по имени (маске)?**  
    `Command` → `Find file` (`<F9>`, `<C>`, `<F>`), ввести маску, убрать галочку `Search for content`.

17. **Как найти текст внутри файлов?**  
    В окне поиска поставить галочку `Search for content` и ввести искомый текст.

18. **Как создать символическую ссылку?**  
    Выделить файл, нажать `File` → `Symlink` (или `<F9>`, `<F>`, `<S>`), указать имя ссылки.

19. **Как узнать размер (занимаемое место) каталогов?**  
    Выбрать `Command` → `Show directory sizes` (или `<Ctrl+Space>`).

20. **Как подключиться к FTP-серверу?**  
    `Left` → `FTP link` (`<F9>`, `<L>`, `<P>`), ввести адрес сервера.

21. **Как подключиться к SFTP-серверу по SSH?**  
    `Left` → `Shell link` (`<F9>`, `<L>`, `<S>`), ввести адрес сервера.

22. **Как увидеть результат команды, введённой во встроенную командную строку?**  
    Ввести команду, затем нажать `<Ctrl+O>` для временного выхода из MC и просмотра вывода.
