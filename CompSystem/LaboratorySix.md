# Контрольные вопросы

## 1. Программы для тестирования видеокарт
Программы делятся на три типа в зависимости от цели проверки:

    Стресс-тесты (проверка стабильности и охлаждения):
        FurMark («Бублик»): Самый известный инструмент для максимального прогрева GPU и проверки системы охлаждения.
        OCCT: Считается одной из лучших программ для поиска ошибок видеопамяти (VRAM) и проверки цепей питания.
        MSI Kombustor: Современная альтернатива FurMark с поддержкой новейших технологий.
    Бенчмарки (оценка производительности в баллах):
        3DMark: Индустриальный стандарт для оценки игровой мощности. Включает тесты для разных разрешений и технологий (например, трассировки лучей).
        Unigine Superposition / Heaven: Популярные бенчмарки на движке Unigine для оценки реальной игровой производительности.
        Geekbench: Кроссплатформенный тест для быстрой проверки вычислительной мощности видеокарты.
    Информационные и инструменты мониторинга:
        GPU-Z: Главная программа для просмотра характеристик: частот, типа памяти, версии драйверов и температур.
        MSI Afterburner: Используется для разгона, настройки лимитов мощности и вывода показателей (FPS, температура) прямо поверх игры. 

## 2. Программы для тестирования мониторов
Тестирование мониторов направлено на выявление битых пикселей, проверку цветопередачи и равномерности подсветки:

    Специализированный софт:
        AIDA64: Содержит встроенный модуль «Тест монитора» с набором калибровочных таблиц и заливок цветом.
        TFT Monitor Test: Утилита для проверки скорости отклика матрицы и поиска артефактов изображения.
        PassMark MonitorTest: Платное решение для профессиональной калибровки и проверки четкости текста.
    Онлайн-сервисы (самый быстрый способ):
        Monteon / ToolFox: Позволяют прямо в браузере запустить тесты на битые пиксели, градиенты, мерцание и динамику.
        EIZO Monitor Test: Один из самых качественных онлайн-инструментов для детальной проверки качества матрицы.
    Встроенные средства:
        В Windows 10/11 есть инструмент «Калибровка цветов», доступный через параметры экрана, который помогает настроить гамму и баланс белого «на глаз».
