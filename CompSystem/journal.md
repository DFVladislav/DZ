
| Вид тестирования | Параметр тестирования | Описание проверки |
|---|---|---|
| **Функциональное** (Functional Testing) | Читабельность текста | Проверяет, читабельны ли статьи и заголовки, нет ли смазанных букв. |
| | Наличие всех страниц/статей | Проверяет, содержит ли журнал все заявленные материалы и страницы по оглавлению. |
| | Правильность нумерации страниц | Проверяет последовательность нумерации страниц. |
| **Юзабилити** (Usability Testing) | Удобство чтения | Оценивает, насколько комфортно держать журнал в руках, перелистывать страницы, воспринимать информацию. |
| | Качество переплета | Проверяет, не распадается ли журнал при активном использовании. |
| | Поиск информации | Проверяет, легко ли найти нужную статью или раздел по содержанию. |
| **Тестирование производительности** (Performance Testing) | Долговечность обложки | Проверяет, как быстро изнашивается обложка (царапины, заломы). |
| | Срок годности | Оценивает, как долго журнал сохраняет товарный вид (выцветание красок, старение бумаги). |
| | Устойчивость к влаге/свету | Проверяет, что произойдет, если на журнал попадет вода или он будет находиться на солнце. |
| **Негативное тестирование** (Negative Testing) | Попытка порвать страницу | Проверка прочности бумаги на разрыв. |
| | Проверка на аллергены | Тестирование реакции на типографскую краску или бумагу. |
| | Засунуть в воду | Проверка устойчивости к экстремальным условиям. |
| **Интеграционное** (Integration Testing) | Совместимость бумаги и чернил | Проверка качества печати на конкретном типе бумаги. |
| | Работа с инструментами для заметок | Проверка возможности писать ручкой/карандашом на страницах журнала. |
| **Тестирование документации** (Documentation Testing) | Правильность выходных данных | Проверяет корректность указания тиража, данных типографии, ISBN. |
| **UI Testing** | Цветопередача | Проверяет, соответствуют ли цвета изображений и дизайна ожиданиям или макету. |
