

| Вид тестирования | Параметр тестирования | Описание проверки |
|---|---|---|
| **Beta Testing** | Отзывы читателей | Сбор обратной связи от тестовой группы читателей перед массовым тиражом. |
| **Automated Testing** | Автоматическая проверка верстки | Использование ПО для проверки верстки, переносов и опечаток. |
| **Validation** | Соответствие ожиданиям читателей | Проверка, соответствует ли журнал заявленной тематике и ожиданиям целевой аудитории. |
| **Verification** | Правильность фактов/данных | Проверка достоверности изложенных фактов, цитат, статистики. |
| **Security Testing** | Безопасность использования | Проверка на наличие острых краев (скрепки, скобы), токсичности краски/бумаги (съедим, отравимся?). |
| **Negative Testing** | Умышленная порча | Попытки порвать, намочить, поджечь, засунуть в розетку. |
| **Stress Testing** | Предельные нагрузки | Проверка на прочность при сгибании, падении, тряске в поезде, использовании в экстремальных условиях. |
| **Documentation Testing** | Инструкция по использованию | Проверка наличия и понятности выходных данных, рекомендаций по утилизации. |
| **Performance Testing** | Долговечность | Оценка срока службы, количества прочтений до потери товарного вида. |
| **UI Testing** | Внешний вид (UI/UX) | Оценка дизайна, цветовой схемы, читабельности шрифтов, наличия иллюстраций. |
| **Entry Point Testing** | Точка входа | Проверка того, как журнал попадает к читателю (куплен новым, уже был в использовании, неизвестен издатель). |
| **Black Box Testing** | Тестирование вслепую | Проверка журнала без знания процесса его создания (например, без макета верстки). |
| **Functional Testing** | Основные функции | Проверка того, пишет ли журнал (если это блокнот), можно ли стереть написанное (если карандашом). |
| **Compatibility Testing** | Совместимость | Проверка совместимости с различными инструментами (ручки, карандаши, маркеры), местами хранения (сумка, рюкзак, полка). |
| **Component Testing** | Тестирование компонентов | Отдельная проверка качества бумаги, качества чернил, качества обложки. |
| **Integration Testing** | Интеграция компонентов | Проверка качества сборки: не отваливается ли обложка от блока, не выпадают ли страницы. |
| **Usability Testing** | Удобство использования | Оценка того, насколько удобно читать, переносить, хранить журнал. |
