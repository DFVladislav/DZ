# Настройка и обновление ОС Windows

**Цель:** Изучить принципы настройки и обновления ОС Windows.

## Классификация обновлений по приоритету

Обновления Windows делятся на четыре основные категории:

1.  **Важные обновления**
    *   **Суть:** Обеспечивают защиту, безопасность и надежность системы.
    *   **Установка:** Должны устанавливаться немедленно; через *Windows Update* обычно устанавливаются автоматически.
2.  **Рекомендуемые обновления**
    *   **Суть:** Устраняют некритические проблемы и улучшают общую производительность.
    *   **Установка:** Могут устанавливаться автоматически (в зависимости от настроек).
3.  **Необязательные обновления**
    *   **Суть:** Включают драйверы и стороннее ПО от Microsoft для оптимизации работы.
    *   **Установка:** Выполняется пользователем вручную.
4.  **Основные обновления**
    *   Крупные функциональные изменения системы.

## Типы обновлений по назначению

*   **Обновление безопасности** — исправление уязвимостей продуктов. Ранжируются по степени опасности: *критическая, важная, средняя или низкая*.
*   **Критические обновления** — исправление серьезных ошибок в программах, не связанных напрямую с безопасностью.
*   **Пакеты обновлений (Service Packs)** — проверенные сборные наборы, включающие в себя все виды исправлений, накопленные с момента выхода продукта. Могут изменять дизайн и функционал.

**2 Отключение обновлений Windows 10**

**2.1.** Запустите редактор локальной групповой политики (нажать `Win+R`, ввести `gpedit.msc`).

**2.2.** Перейдите к разделу **«Конфигурация компьютера»** — **«Административные шаблоны»** — **«Компоненты Windows»** — **«Центр обновления Windows»**. Найдите пункт **«Настройка автоматического обновления»** и дважды кликните по нему.
